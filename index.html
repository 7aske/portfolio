HTML5, CSS3 and JS are fundamental technologies for every single web developer hence im not going to go in-depth about them since I feel that it is a waste of time and website space to even note them down. Okay, okay jQuery as well *sighs*.

When I started learning to code I was aspiring to become a full-stack developer. I found javascript really intuitive, fun and easy to learn when I was deciding which backend technology to go for node.js was a logical choice. I learned how to make my own servers that I hosted on Heroku. Feeling of actually typing an address in my browser and seeing my own website on the web was amazing!

Next obvious step was to learn how to save my dynamic website data so that I can start developing more interesting apps. I experimented with local storage databases like lowdb and manually writing data to JSON files with node but those methods haven't matched my demands. Having similar syntax and learning resources being widely available I chose MongoDB as my go-to cloud database. Node.js mongoose drivers provided an easy to use system to develop dynamic applications.

Meanwhile, as I was doing my thing I stumbled upon a nifty little framework called electron.js. It felt amazing to actually see my then web applications being installed and ran (somewhat) natively on my PC.

When I realized that one of the requirements for a current day web developer is to develop in one of the popular UI frameworks I had another decision to make. Since I've read that React can be used to develop android/iOS applications as well I chose it over Angular and Vue. Redux seems to be an irreplaceable tool when developing with React.
